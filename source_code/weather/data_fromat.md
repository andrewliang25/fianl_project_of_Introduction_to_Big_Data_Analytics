## 氣象觀測資料
|  欄位 	| 說明                           	|
|:-----:	|--------------------------------	|
| stationName	| 測站名稱                  	|
| time  	| 時間                  	|
| WDIR  	| 風向，單位 度                  	|
| WDSD  	| 風速，單位 公尺/秒             	|
| TEMP  	| 溫度，單位 攝氏                	|
| HUMD  	| 相對濕度，單位 百分比率(0-1.0) 	|
| PRES  	| 氣壓，單位 百帕                	|
